# chamada-de-senha
HTML para chamada de senhas básico
Com opção de chamada N para senha normal (tecla n), P para senha de prioridade (tecla p) e E para pegar exames (tecla e). Quando a senha é chamada uma voz avisa qual senha foi chamada. Ao final temos a tecla r para zerar todos os contadores.
